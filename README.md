## 📌 Overview

This is my personal portfolio website built using modern web technologies.  
It showcases my projects, skills, and contact information in a clean and responsive design.

The website is fully responsive and optimized for performance.

---

## 🚀 Tech Stack

- ⚛️ React
- ⚡ Vite
- 🎨 Tailwind CSS
- 🌐 Netlify (Deployment)
- 🖥️ Node.js

---

## ✨ Features

- Responsive design (Mobile + Desktop)
- Modern UI with Tailwind CSS
- Fast performance powered by Vite
- Clean component-based architecture
- Serverless support (Netlify functions)
- Easy to customize content

## ⚙️ Installation

1️⃣ Clone the repository
```
bash
git clone https://github.com/Aditya-Singh-16/your-repo-name.git
```
2️⃣ Navigate to project folder
```
cd your-repo-name
```
3️⃣ Install dependencies
```
npm install
 or
pnpm install

```
▶️ Run Locally
```
npm run dev
Then open:
http://localhost:5173
```
📦 Build for Production
```
npm run build
```
